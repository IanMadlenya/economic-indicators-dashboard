About

This dashboard provides an at-a-glance overview of the San Jose economy. The goal is to provide city leaders, businesses and residents with a convenient, reliable source of economic data, enabling informed discussion and decision-making.

The indicators will be updated on a quarterly basis. To subscribe to announcements about data releases and new features, send us an email.

Help Us Improve The Dashboard

This site is a work in progress. We want to hear what datasets, information and other features would make this site most useful to you. Email Michelle Thong in the San Jose Office of Economic Development to share your feedback.

If you're comfortable using Github, you can also report a bug or file a feature request on our issue tracker.

How This Dashboard Was Made

This dashbord was inspired by similar sites for Metro Denver, the San Diego region and Chicago.

The design and implementation of the San Jose dashboard was made possible by a partnership between Code for San Jose volunteers and the San Jose Office of Economic Development. The core team consisted of:

  • Michelle Thong - Project Lead, Code for San Jose | San Jose Office of Economic Development
  • Laurie Reynolds - Lead Developer, Code for San Jose
  • Danny Shapira - Front-End Developer, Code for San Jose

The indicator data are compiled by the Office of Economic Development using public and private sources. Data sources for each indicator are revealed by positioning a cursor over the information symbol in the bottom left corner of each box.

Make Your Own Dashboard

The San Jose Economic Indicators dashboard is an example of a civic innovation project that leverages open source code, open data and government-community partnerships.

The code is available on Github, including documentation of the technologies used and how to get started. All data used for this site can be downloaded from the City of San Jose's Open Data Portal.
